14 ##CUnit Compilation Notes
15 
16 When builing the test unit, there was an issue linking the CUnit header files to t_lib.c file.
17 The switch \b -lcunit was used, however the location of the switch was wrong.
18 The \b -lcunit switch MUST preceed the src files in the compile command.
19 
20 Example:
21 
22  gcc -lcunit t_lib.c lib.c -o test.out
23 
24 ##Makefile Notes
25 
26 A make file is used to build, execute, run test, and generate Doxygen files.
27 
28 Make commands:
29 
30  make: Builds the executable adder.out
31  run: Runs adder.out
32  makeTest: Builds the executable t_lib.out
33  runTest: Runs t_lib.out
34  docs: Generates Doxygen documentation
35  viewDocs: View the generated documentation in Firefox
36  clean: Removes contents of builds and deletes the docs file
